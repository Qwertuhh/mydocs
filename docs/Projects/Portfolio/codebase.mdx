## Overview

The portfolio is an site to showcase yourself and your work. In my case I created the my portfolio with
React-based single-page application built with TypeScript, Vite, and modern web technologies.
It features interactive elements, custom audio effects, and a responsive design with smooth animations.


### Key Technologies

- **React 18** with TypeScript
- **Vite** for build tooling
- **React Router** for navigation
- **TailwindCSS** for styling
- **Custom Audio System** with Web Audio API

## Application Structure

### Main Entry Points

#### `src/main.tsx`

The application's entry point that:

- Sets up React Strict Mode
- Configures React Router with two routes:
  - `/static` - Static page version
  - `/*` - Main interactive portfolio
- Wraps the app with AudioProvider for global audio context
- Renders the root element

#### `src/App.tsx`

Main application component that:

- Manages user interaction state
- Implements lazy loading for performance
- Orchestrates all major components
- Integrates custom cursor and audio systems
- Uses Suspense with loading fallbacks

### Component Hierarchy

```
App
├── CustomCursor
├── InteractionOverlay
├── AudioControl (conditional)
├── Navbar (lazy)
├── HeroComponent (lazy)
├── AboutMe (lazy)
├── Projects (lazy)
└── Footer (lazy)
```
